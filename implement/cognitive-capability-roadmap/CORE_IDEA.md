# Cognitive Capability Roadmap

## Problem

Mneme has a strong deterministic brain scaffold, but local AI models are not yet connected. The project also needs a way to compare cognitive growth over time without making vague or misleading claims about sentience or biological equivalence.

## Desired Outcome

Create a durable roadmap for:

- local model integration,
- model-safe cognition boundaries,
- benchmark-driven capability growth,
- animal-reference capability comparison,
- a long-term human-brain-equivalent functional target,
- a clear gate before physical embodiment resumes.

## User Value

The roadmap lets future implementation stay focused on building the brain first: local speech, vision, memory, attention, model-backed reasoning, evaluation, and adaptation before hardware.

## Affected Systems

- architecture roadmap,
- repo status,
- backlog,
- future local model runtime,
- future cognitive benchmark harness,
- future UI observability.

## Constraints

- No runtime behavior changes in this planning task.
- No new dependencies.
- Do not claim biological sentience.
- Preserve the architecture rule that models do not bypass memory, executive intent, skills, or safety.

## Non-Goals

- Implementing local model adapters now.
- Choosing one required LLM runtime.
- Adding cloud dependency.
- Starting physical embodiment.
