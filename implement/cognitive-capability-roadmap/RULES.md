# Rules

- Local models are cognitive tools, not the whole brain.
- Models may propose memory candidates but must not silently confirm facts.
- Model outputs remain `model_inferred` unless user-confirmed.
- No direct model-to-actuator control.
- No cloud dependency as a hard requirement.
- No unrestricted self-modification.
- No biological sentience claims from benchmark scores.
- Capability comparisons must be behavioral and evidence-backed.
