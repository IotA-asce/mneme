# M8 Cognitive Evaluation Rules

- Benchmarks must run locally with fake/runtime paths and must not require Ollama, real devices, or physical hardware.
- Benchmark runs use temporary databases by default.
- Capability ladder output is behavioral evidence only, never animal equivalence or sentience.
- Turn classification may feed executive/dialogue behavior but must not bypass memory, safety, or intent boundaries.
- Correction, contradiction, and forget turns must not silently mutate durable memory in this slice.
- Model wording remains optional and cannot create confirmed memories.
