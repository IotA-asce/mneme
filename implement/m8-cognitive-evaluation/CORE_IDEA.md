# M8 Cognitive Evaluation Core Idea

## Problem

Mneme can use a local model as a wording layer, but subjective chat quality is not enough to know whether the brain loop is improving.

## Desired Outcome

Add deterministic benchmarks, turn classification, memory review, and conservative capability evidence so Mneme can explain what it used and measure whether changes improve continuity without increasing unsupported memory claims.

## Value

This makes the project less like a chatbot by routing turns through explicit intent categories and by requiring benchmark evidence before claiming cognitive progress.

## Constraints

- No cloud models.
- No embeddings or vector database.
- No model-owned memory writes.
- No physical hardware or actuator work.
- Correction and forget requests are proposals only in this slice.

## Non-Goals

- Natural-language beauty.
- Biological sentience claims.
- Autonomous correction approval.
- Long-running daily-driver soak replay.
