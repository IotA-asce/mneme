# Rules

- Local models are tools behind Mneme's cognition boundaries, not the whole brain.
- Do not let model output bypass memory retrieval, executive intent, skill goals, or safety.
- Do not treat model output as user-confirmed truth.
- Do not auto-download model files or Ollama models.
- Keep all automated tests fake-backed.
- Use service-managed registry records for Ollama models and file-managed records for `.local/models/`.
- Preserve deterministic fallback behavior until a model dialogue realizer is explicitly implemented.
