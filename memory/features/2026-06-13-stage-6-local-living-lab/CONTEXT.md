# Context

After Stage 5, Mneme had virtual speech, avatar state, virtual skills, local TTS command support, and deterministic barge-in handling. The previous roadmap still pointed next toward physical embodiment.

The owner clarified the desired trajectory: build the brain first on the current computer, using local camera, microphone, speaker, local models, memory, attention, and virtual presence. Physical hardware should wait until the brain loop can be tested daily and behaves coherently.

This change makes Stage 6 the Local Living Lab and keeps all physical actuation, GPIO, serial, PWM, ROS control, firmware flashing, and motors out of scope.
