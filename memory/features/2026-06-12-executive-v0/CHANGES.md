# Changes

- Added `ExecutiveMode`, `ExecutiveIntentType`, and `ExecutiveIntent`.
- Added deterministic `Executive` rule engine.
- Added initial intents for look, listen, response, memory, idle, freeze, and degraded mode.
- Added priority and preemption handling.
- Published runtime `executive_intent` events only.
- Added tests for priority, preemption, degraded mode, fallback intents, serialization, and runtime publication.
- Documented the subsystem in `docs/executive/EXECUTIVE_V0.md`.
