# Context

Before Stage 5, Mneme could run the local virtual-head loop, process typed or adapter-provided speech transcripts, remember facts, retrieve memory, generate executive intent, and produce deterministic dialogue plans. The loop still rendered text directly and had no virtual skill/status layer for speech, avatar state, or interruption behavior.

The master roadmap requires proving conversational timing and skill contracts before physical embodiment. This work keeps that proof local and deterministic.
