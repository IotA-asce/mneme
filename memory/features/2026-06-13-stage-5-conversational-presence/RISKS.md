# Risks

- Local TTS quality and latency depend on host-installed tools.
- Graphical avatar rendering is not implemented; Stage 5 exposes state only.
- Speaker device routing is a placeholder contract, not a verified audio-routing implementation.
- Barge-in is deterministic at the event level; real endpointing depends on the Stage 4 speech adapter.
- No physical safety conclusions should be drawn from virtual skill success.
